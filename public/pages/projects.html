The Idioma project was designed to be a all-in-one language application. Using multiple libraries Idioma can define, translate, and connote a word or phrase. This was a group project with Mick handling the HTML and CSS elements as well as external libraries.

FoodDex
FoodDex was made with the intention to be an online food/recipe wall. With use of libraries such as Node.js, Express, AWS S3, etc. the project allows anyone to post a recipe, with a photo, description, and recipe for everyone to view. It is also equipped with a useful search bar with the ability to search without specifics. This was a group project with Mick taking on the HTML, CSS, AWS S3, and final touches of the JS.
